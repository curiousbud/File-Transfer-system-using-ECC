# Qubix       <img alt="GitHub" src="https://img.shields.io/github/license/curiousbud/File-Transfer-system-using-ECC">


Qubix is a secure file transfer system built with Django, designed as a major project with advanced cryptographic features using Elliptic Curve Cryptography (ECC).

## Qubix Features

<h4>Core Features:</h4>
<ol>
    <li>Secure user authentication and login system.</li>
    <li>File upload functionality with ECC encryption.</li>
    <li>User can view their uploaded files securely.</li>
</ol>

<h4>Advanced Features:</h4>
<ol>
     <li>User search and profile viewing capabilities.</li>
     <li>Secure file sharing between users with cryptographic protection.</li>
     <li>Shared files integration with advanced security measures.</li>
</ol>

<h4>Additional Features:</h4>
<ol>
    <li>In users profile user can set his/her profile picture.</li>
    <li>Users can download other users uploaded files.</li>
    <li>The user can upload any type of files such as images, videos, text files and also different types of programs like python code, java code, etc.</li>
</ol>

<h2>About Qubix:</h2>

Qubix is an advanced secure file transfer system that leverages Elliptic Curve Cryptography (ECC) to provide robust security for file sharing and storage. Built with Django and modern web technologies, Qubix offers a user-friendly interface while maintaining enterprise-level security standards.

<h2>Key Technologies:</h2>
<ul>
    <li>Django Framework</li>
    <li>Elliptic Curve Cryptography (ECC)</li>
    <li>Bootstrap 4</li>
    <li>django-crispy-forms</li>
    <li>Pillow (PIL)</li>
    <li>SQLite Database</li>
</ul>

<h2>Note :</h2>

<b>This project is designed as a major project demonstrating secure file transfer capabilities. The Secret_Key is included for educational purposes and easy setup. You can use this project as a foundation for your own secure file transfer applications.</b>

<h2>Usage :</h2>

    python qubix/manage.py makemigrations

    python qubix/manage.py migrate

    python qubix/manage.py runserver
    
   In your web browser enter the address : http://localhost:8000 or http://127.0.0.1:8000/
